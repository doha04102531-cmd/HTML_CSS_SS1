Giới thiệu về môn Frontend

Frontend là gì?

Frontend là phần quan trọng của phát triển web, bao gồm mọi thứ mà người dùng có thể nhìn thấy và tương tác trực tiếp trên trang web

Các công nghệ Frontend phổ biến

  • HTML (HyperText Markup Language): Ngôn ngữ đánh dấu siêu văn bản, dùng để cấu trúc nội dung trên web.
  • CSS (Cascading Style Sheets): Ngôn ngữ định kiểu, dùng để tạo kiểu dáng và bố cục cho trang web.
  • JavaScript: Ngôn ngữ lập trình, dùng để tạo các tương tác động và hiệu ứng trên trang web.
